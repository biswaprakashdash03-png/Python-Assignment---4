# Python Assignment-4 Programs 🐍

This repository contains beginner-friendly Python programs focused on loops, conditional statements, arithmetic operations, and user input handling.

## Programs Included

* Print Even Numbers from 1 to 10
* Print Odd Numbers from 1 to 10
* Sum of Even Numbers from 1 to 20
* Sum of Odd Numbers from 1 to 20
* Sum of Unit Digits of Even Numbers (11–20)
* Student Grade Calculator

## Objective

These programs help beginners understand and practice fundamental Python concepts such as loops, conditions, arithmetic calculations, and user input processing. The assignment is designed to improve logical thinking, problem-solving abilities, and programming skills through simple and practical exercises.
